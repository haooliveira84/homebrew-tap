# homebrew-tap

Homebrew formulae maintained by [@haooliveira84](https://github.com/haooliveira84).

## Usage

```sh
brew tap haooliveira84/tap
brew install kubernetes-tools
```

## Available formulae

| Formula | Description | Upstream |
|---------|-------------|----------|
| `kubernetes-tools` | Scripts that simplify daily Kubernetes operations | [haooliveira84/kubernetes-tools](https://github.com/haooliveira84/kubernetes-tools) |

## Updating

The formulae here are bumped automatically by the upstream release pipelines.
See [`kubernetes-tools/.github/workflows/release.yml`](https://github.com/haooliveira84/kubernetes-tools/blob/master/.github/workflows/release.yml).
